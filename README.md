# OPG Mraz - Online Trgovina

Web stranica za OPG Mraz s internetskom trgovinom za prodaju svježih poljoprivrednih proizvoda.

## Tehnički detalji

Ovaj projekt je izgrađen koristeći [Next.js](https://nextjs.org) s TypeScript-om, Tailwind CSS-om i App Router-om.

## Pokretanje

Prvo, instalirajte dependencies:

```bash
npm install
```

Zatim, pokrenite development server:

```bash
npm run dev
```

Otvorite [http://localhost:3000](http://localhost:3000) u vašem browseru.

## Funkcionalnosti

- Katalog proizvoda
- Košarica za kupovinu
- Proces plaćanja s obrascem za dostavu
- Responzivni dizajn
